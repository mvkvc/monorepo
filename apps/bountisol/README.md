# bountisol

Fully transparent on-chain bounties for Solana.

## Links

- https://bounti.sol
- https://bountisol.xyz

## Components

```bash
.
├── app # Full-stack web application
├── programs # Solana programs
└── sh # Shell scripts
```

## Developing

### Requirements

- [mise](https://mise.jdx.dev/getting-started.html)
- [rust](https://rustup.rs)

### Setup

- `cp .env_ .env`
- `mise trust`
- `mise install`

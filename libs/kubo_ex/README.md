Elixir library for interacting with a Kubo IPFS node.

## Installation

The package can be installed by adding `kubo_ex` to your list of dependencies in `mix.exs`:

```elixir
def deps do
  [
    {:kubo_ex, git: "https://github.com/mvkvc/kubo_ex.git"}
  ]
end
```
